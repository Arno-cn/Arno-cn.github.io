Python基础教程-基础知识以及准备工作

发表于 2017-07-27 | 分类于 Python


Python的安装和基础知识。

在这个博客中我会简单地告诉你Python是什么，为什么你应该使用它，以及如何使用。

开场白

那么什么是Python？简单地用官方语言来说，那就是Python是“一种解释型的，面向对象的，带有动态语义的高级程序设计语言”,有点晦涩难懂，但是最重要的是，Python是一种想让你在编程实现自己想法时感觉不那么碍手碍脚的程序设计语言。你可以花较少的代价实现想要的功能，并且编写的程序清晰易懂。

安装Python

在开始编程前，需要安装一些新软件。下面简要介绍如何下载和安装Python。如果想直接跳到安装过程而不看详细的向导，可以直接访问网址Python下载。

因为系统原因，下面只介绍如何在Windows系统中安装Python，Linux及Unix版本的安装方法请自行上网搜索。

Windows安装

要在Windows系统中安装Python，请参照下面的步骤进行：

  1. 打开Web浏览器，访问Python官网。
  2. 点击Download链接。
  3. 在这里可以看到几条链接，名字类似于Python2.x.y或者Python2.x.y Windows installer。
    点击Windows installer链接并且下载安装文件。
  4. 将Windows Installer文件放在计算机的任何位置均可，比如C：\download\python-2.x.y.msi。
  5. 在Window 资源管理器中双击运行下载的文件，开启Python的安装向导。只需要接受默认设计，等到安装程序完成，你就能使用Python了。

假如安装一切正常，应该能在Windows的开始菜单中找到新安装的Python程序。按照开始→所有程序→Python→IDLE的步骤运行Python集成开发环境（ Python Integrated Development Environment,IDLE）。

你现在应该看到一个这样的窗口，至此Window下的Python安装就已经完成了。

算法是什么

在开始编程之前，首先要知道的是什么是计算机程序设计。简单的说，它就是告诉计算机该做什么，计算机可以做很多事情，但是它就像婴儿一样，不知道自己该怎么做。程序员通过使用计算机能够理解的语言——算法来告诉计算机具体的细节。

简单地来说，算法就像是步骤，也就是对如何做一件事的详细描述，比如：

出门：

首先，穿好衣服和鞋子；

如果快要下雨了，带上一把雨伞；

然后，出门；

每30分钟检查一次自己的鞋带是否开了；

这个步骤不是特别有趣，但是它的组成结构还是有点讲究的。它包括按照顺序执行的指令（ ”穿好衣服和鞋子“ ），考虑特定条件的指令(”如果快要下雨了，带上一把雨伞“)，还有一些是需要重复的(“每30分钟检查一次鞋带是否开了”)。
步骤和算法都包括一些材料（对象，物品），和指令（语句），接下来我们来看看一些Python的“材料”。

注释

Python中使用#号来作为注释符号，出现在其右边的一切内容都会被忽略掉。
所以它的作用就很明显了，用来说明自己的程序某个地方是用来做什么的，这样当别人看你的代码时更容易理解。
