发表于 2017-07-23 | 分类于 博客



   写作缘由： 分享搭建博客网站的经验

说明

  • 本文使用Github和Hexo来搭建博客网站
  • 使用工具：Git , Node.js , Hexo

步骤

  1. 购买域名。
  2. 创建Git账户和Github仓库。
  3. 安装Nghode.js。
  4. 创建本地文件。
  5. 推送网站。
  6. 网址解析。

购买域名

 域名是便于记忆和沟通的一组服务器的地址，比如百度的域名是baidu.com。域名有许多后缀，常见的有com,cn,net,org等，可以根据你自己的喜好来选择域名后缀。申请域名的地方很多，我选择的是阿里云注册账号之后购买一个域名，这也是所有步骤中唯一花钱的地方。

网址解析

 如果每次访问博客都需要输入github.io太麻烦了，于是我们使用域名解析的方式将购买的域名指向我们的Github仓库，这样就能通过输入购买的域名来访问我们的博客了。

 登录到阿里云，进入域名控制台，找到你的个性化域名，进入解析：

然后添加解析：

 包括添加三条解析记录，192.30.255.112是GitHub的地址，你也可以ping你的 http://xxxx.github.io 的ip地址，填入进去。第三个记录类型是CNAME，CNAME的记录值是：你的用户名.github.io 这里千万别弄错了。第二步，登录GitHub，进入之前创建的仓库，点击settings

 往下拉，设置Custom domain，输入你的域名

 点击save保存。第三步，进入本地博客文件夹 ，进入blog/source目录下，创建一个记事本文件，输入你的域名，对，只要写进你自己的域名即可。如果带有www，那么以后访问的时候必须带有www完整的域名才可以访问，但如果不带有www，以后访问的时候带不带www都可以访问。所以建议，不要带有www。

 保存，命名为CNAME，注意要保存成所有文件，而不是txt文件。
